Álgebra Lineal y Cálculo de Tensores

El álgebra lineal no es solo el estudio de tablas de números; es el marco mediante el cual entendemos cómo transformar y deformar espacios. Cuando añadimos el cálculo a esta mezcla, obtenemos las herramientas necesarias para optimizar funciones complejas en dimensiones masivas.

La Jerarquía de los Tensores

Un tensor es un objeto matemático multilineal definido por su rango.

Rango Nombre Significado Geométrico

0

Escalar ($k$)

Magnitud pura en un punto.

1

Vector ($\vec{v}$)

Dirección y magnitud en el espacio.

2

Matriz ($A$)

Transformación lineal entre espacios.

$n$

Tensor de orden $n$

Generalización multidimensional.

En computación, el rango de un tensor se suele llamar su número de ejes o dimensiones (rank o ndim).

Espacios, Kernel e Imagen

Dada una matriz $A$, esta define dos subespacios fundamentales:

  • Imagen o Espacio Columna ($Im(A)$): El conjunto de todos los vectores que pueden ser alcanzados por la transformación $A\vec{x}$. Su dimensión es el rango de la matriz.

  • Kernel o Espacio Nulo ($Ker(A)$): El conjunto de vectores $\vec{x}$ que $A$ "aplasta" hacia el cero ($A\vec{x} = \vec{0}$).

Si el Kernel contiene vectores distintos de cero, la transformación no es inyectiva; es decir, hay pérdida de información irreversible.

Broadcasting (Difusión de Tensores)

El Broadcasting es una regla matemática y computacional que permite realizar operaciones elemento a elemento entre tensores de distintas formas (shapes).

Dos dimensiones son compatibles para el broadcasting si: 1. Son iguales. 2. Una de ellas es $1$.

Tensor A (Shape: 3 x 4)
Tensor B (Shape:     4)
-----------------------
A + B (Resultado: 3 x 4)

Explicación: La dimensión de $B$ se "difunde" virtualmente a lo largo de las filas de $A$ para que la operación sea posible.

De la Teoría a la Práctica: Nx

En Nx, la diferenciación automática y el broadcasting son ciudadanos de primera clase.

import Nx
alias Nx.LinAlg

# Ejemplo de Broadcasting
a = Nx.tensor([[1, 2], [3, 4]])
b = Nx.tensor([10, 20])
Nx.add(a, b) # [[11, 22], [13, 24]]

# Diferenciación Automática (Gradiente)
f = fn x -> Nx.sum(Nx.pow(x, 2)) end
gradiente_f = Nx.grad(f)

# Evaluar el gradiente en un punto
x = Nx.tensor([1.0, 2.0, 3.0])
gradiente_f.(x) # [2.0, 4.0, 6.0]

Nx utiliza modos de compilación para calcular estos gradientes de forma extremadamente eficiente, permitiendo que el compilador optimice la cadena de derivadas (regla de la cadena) antes de la ejecución.
